# Usage (Windows PowerShell):
# 1) Create a Python 3.10 venv and activate it (recommended):
#    py -3.10 -m venv venv
#    .\venv\Scripts\Activate.ps1
# 2) Run the automated bootstrap (creates venv if missing and installs deps):
#    .\setup_env.bat         # or run manually the commands below
#
# NOTE: PyTorch (torch) is installed separately because GPU wheels must match
# your CUDA/driver. After activating the venv choose one of the following:
#  - GPU (example for CUDA 12.x / wheel `cu121`):
#      pip install --index-url https://download.pytorch.org/whl/cu121 --upgrade --force-reinstall torch torchvision torchaudio
#  - CPU-only fallback:
#      pip install --index-url https://download.pytorch.org/whl/cpu torch torchvision
#  - Conda (recommended for GPU users with conda/miniconda):
#      conda install pytorch torchvision torchaudio pytorch-cuda=12.1 -c pytorch -c nvidia
#
# After installing PyTorch run:
#    python diagnose_env.py
#    python gui_main.py
